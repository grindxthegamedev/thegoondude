# Public Assets

## Usage

Assets are accessible at `/filename`:
- `/next.svg` → Next.js logo
- `/globe.svg` → Globe icon

## Notes

- Keep static assets minimal
- Large media (videos) should use Firebase Storage
- SVGs preferred for icons (smaller, scalable)
