# Sites Directory Pages

Directory listing pages for site browsing.

## Routes

| Route | Description |
|-------|-------------|
| `/sites` | Main directory with all sites |
| `/sites/[category]` | Category-filtered listings |

## Features

- **Sort Options**: Rating, date added, alphabetical
- **Category Filter**: Dynamic category pages
- **Site Listings**: 411-style row components
- **Responsive Grid**: Adapts to viewport

## Files

- `page.tsx` - Main listing with filters
- `page.module.css` - Listing styles
- `[category]/page.tsx` - Category-specific listing
- `[category]/layout.tsx` - Category layout with breadcrumbs
